## 2026 电子树脂材料行业深度互动报告

这是一个单文件的交互式行业研究报告页面，围绕 **电子树脂材料** 行业构建，采用现代前端展示方案，适合作为研究分享或投研路演的可视化载体。

### 功能概览

- **执行摘要**：报告总体结论和行业核心逻辑。
- **行业概览**：全球与中国电子树脂市场规模对比与预测，使用柱状图展示。
- **核心材料解析**：环氧树脂、PPO、PTFE、BMI 等核心树脂的性能与应用，对应交互式标签页与雷达图。
- **驱动力分析**：AI 服务器、800G/1.6T 网络、智能网联汽车等下游需求驱动模块。
- **竞争格局与国产替代**：国产化率趋势折线图与代表性企业案例卡片。

### 技术栈

- **HTML 单页应用**：无需构建工具，直接双击 `index.html` 即可在浏览器中打开。
- **Tailwind CSS CDN**：通过官方 CDN 引入，用于快速构建现代化 UI。
- **Chart.js CDN**：用于绘制定制化的柱状图、环形图、雷达图和折线图。

### 本地运行方式

1. 克隆本仓库：

```bash
git clone https://github.com/your-username/resin-2026.git
cd resin-2026
```

2. 直接在浏览器中打开 `index.html`：

```bash
open index.html   # macOS
# 或在 Windows 上双击文件，或使用浏览器 File -> Open
```

> 若需要避免某些浏览器的本地文件安全限制，可在本地启动一个简单的静态服务器，例如：

```bash
python -m http.server 8000
```

然后访问 `http://localhost:8000/index.html`。
